# Better Shell - BSH

Better Shell is a Rust-based open source project aiming to combine all the nice features of the standard Shell, BASH, ZSH, FISH and more. 

## Contributing to Better Shell

We greatly appreciate all contributions to Better Shell. This is a community-driven project! We aren't very strict when it comes to contributing. Just make sure your code is readable and maintainable. It should also follow the style of the other code in this project.
We are happy to welcome anybody!
